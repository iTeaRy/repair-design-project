## Steps to Setup repair-design-project

1. **Clone the application**

	```bash
	git clone https://github.com/... .git
	cd repair-design-project
	```

2. **Install**
    + npm "https://www.npmjs.com/get-npm"
	```bash
	npm install -g sass
	```

3. **Run**
	```bash
	sass index.scss index.css
	```
